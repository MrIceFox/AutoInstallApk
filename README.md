# AutoInstallApk

----------

一些没有在Google Play上架的app在MIUI国际版(无小米商店)无法自动更新，而酷安市场下载更新之后无法自动安装，需要手动一个个点击安装。
为了解决这个问题，写了一个可以自动批量安装apk的小工具。
